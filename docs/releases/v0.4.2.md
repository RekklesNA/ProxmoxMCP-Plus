# ProxmoxMCP-Plus v0.4.2

Release date: 2026-04-28

## Summary

This release restores and updates the container command execution setup documentation for the
current SSH-backed `pct exec` implementation.

## Changes

- Rebuilt the LXC command execution guide with the current `mcp-agent` plus scoped sudo setup.
- Added command policy, `known_hosts`, host override, and troubleshooting guidance.
- Added README and wiki navigation entries for container command execution.
- Updated the example config to match the recommended non-root SSH user setup.

## Upgrade Notes

- No runtime migration is required.
- Users enabling `execute_container_command` should review the updated guide and confirm their
  `ssh` and `command_policy` config sections match the current examples.
