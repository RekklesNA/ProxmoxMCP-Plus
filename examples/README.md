# Examples

These examples are designed for fast copy-paste usage in three contexts:

- `Claude Desktop` or another MCP client
- `Open WebUI` or another LLM / AI agent interface
- `HTTP/OpenAPI` callers that want concrete request payloads

Available templates:

- [Create a test VM](create-test-vm.md)
- [Roll back a risky change with snapshots](rollback-snapshot.md)
- [Download an ISO and create an LXC](download-iso-and-create-lxc.md)
